# Audio Steganography Project

## 📌 Description
This project hides secret messages inside audio files using Python.

## 🛠️ Technologies Used
- Python
- Wave Library
- Steganography Concepts

## ⚙️ How it Works
- Input: Audio file + secret message
- Process: Message is encoded into audio
- Output: New audio file with hidden data

## 🚀 Features
- Secure message hiding
- Easy to use
- Lightweight
